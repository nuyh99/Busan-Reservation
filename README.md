# 로컬에 스프링 띄우기

1. 클론한 후 터미널에서 프로젝트 최상위 디렉토리로 이동한다.
2. 아래의 명령어를 입력한다.

```bash
sudo docker compose up
```

혹시 `docker.io` 관련 에러가 뜬다면 아래의 명령어를 먼저 실행합니다.

```bash
rm  ~/.docker/config.json
```
